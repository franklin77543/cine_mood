# CineMood - UI/UX 設計概念

## 🎨 設計理念

**核心概念**: "心情驅動的電影探索"  
**設計風格**: 現代簡約 + 電影感 + AI 互動感

### 設計原則
1. **情感優先** - UI 反映不同心情氛圍
2. **對話式交互** - AI 推薦感覺像與朋友聊天
3. **視覺沉浸** - 大圖、漸變、電影感
4. **極簡操作** - 少點擊，多智能

## 🎬 互動設計

### AI 對話互動

**流程**:
```
用戶輸入 → 打字中動畫 → AI 思考動畫 → 結果淡入
```

**動畫效果**:
1. **AI 思考**: 脈衝光環 + "正在分析你的心情..."
2. **意圖解析**: 標籤從中心展開
3. **推薦卡片**: 從下往上彈出，依序延遲
4. **相似度**: 進度條動畫填充

### 微互動

- **卡片 Hover**: 放大 1.05x + 陰影加深
- **按鈕點擊**: 漣漪效果
- **收藏**: 心形填充動畫
- **標籤選擇**: 背景色漸變
- **滾動**: 視差效果

## 🌈 主題變化

### 心情主題切換

當用戶選擇心情時，UI 動態調整：

**輕鬆** (`--mood-happy`):
- 主色: 黃色系
- 背景: 溫暖漸變
- 字體: 圓潤

**療癒** (`--mood-sad`):
- 主色: 藍色系
- 背景: 柔和冷色
- 動畫: 緩慢

**刺激** (`--mood-exciting`):
- 主色: 紅色系
- 背景: 對比強烈
- 動畫: 快速
